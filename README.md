# Student Registration System

This project is a rebuilt and improved version of a university assignment.

It was developed using PHP, MySQL, HTML, and CSS to practice backend development, database integration, and user authentication.

## Features
- User Registration
- Secure Login
- Password Hashing
- Session Management
- Responsive User Interface

## Technologies Used
- PHP
- MySQL
- HTML
- CSS

## Note
This project was originally created as part of a university assignment and was rebuilt with cleaner code, improved design, and better project organization.

Before running the project, rename `db_connect.example.php` to `db_connect.php` and add your own database credentials.
